Perfect! Let’s update the Prescription Workflow for Medicata MVP to include expiry date, patient dashboard storage, and re-order functionality. I’ll break it down step by step.

Medicata Prescription Workflow with Expiry & Reorder (MVP)

1️⃣ Access Prescription During Consultation
Doctor opens the Consultation screen for a patient
Button: Create Prescription / Add Prescription

2️⃣ Drug Selection
Doctor sees a searchable drug database
Search by drug name, category, or brand
Doctor clicks select to add drug to prescription

3️⃣ Prescription Form
For each drug added:
Fields Doctor Must Fill:
Dosage – e.g., 500mg, 10ml
Frequency – e.g., once a day, twice a day
Duration / Number of Days – e.g., 5 days, 10 days
Instructions / Notes – optional (e.g., “Take with food”)
Quantity – number of pills / ml
Prescription Expiry Date – required (e.g., 30 days from issue)
Can add multiple drugs in one prescription
Can edit or remove each drug entry before sending

4️⃣ Prescription Confirmation
Doctor reviews prescription
Click Send to Patient
Optionally: Send to Pharmacy
System Actions:
Generates digital prescription
Saves prescription automatically in patient dashboard
Sends notification to patient & selected pharmacy

5️⃣ Patient Dashboard
Features:
View all prescriptions (active & expired)
Prescription Details: dosage, frequency, duration, instructions, expiry date
Buy Again Button: available when prescription expires
Optionally forward to pharmacy for refill
Expiry Logic:
After expiry date, prescription status = Expired
System can notify patient 3 days before expiry
“Buy Again” creates a new order using same prescription info

6️⃣ Pharmacy Workflow
Receives prescription if doctor sent it
Can update status: pending, fulfilled, delivered, picked up
If prescription expired, pharmacy cannot fulfill unless patient requests new order

7️⃣ Post-Consultation Actions
Doctor:
Prescription saved in consultation record
Can track if patient has used prescription or needs refill
Optionally set follow-up consultation
Patient:
Access prescription anytime in dashboard
See expiry date and instructions
Re-order when expired
